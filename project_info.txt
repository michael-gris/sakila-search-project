Финальный проект по курсу Python

Название: Каталог фильмов с аналитикой

Описание:
Программа позволяет выполнять поиск фильмов из базы данных (MySQL) по ключевым словам или по жанру и году выпуска. Все пользовательские запросы логируются в MongoDB, что позволяет собирать аналитику: какие типы запросов наиболее популярны.

Функциональность:

Подключение к базе данных MySQL (sakila)

Запросы по частям названий фильмов

Запросы по ID жанра и диапазону годов

Подсчёт и отображение топ-5 популярных типов запросов

Логирование всех запросов в MongoDB (с указанием параметров и времени)

Работа через консольное меню

Используемые технологии:

Python 3.12

MySQL (через mysql-connector-python)

MongoDB (через pymongo)

dotenv (переменные окружения)
