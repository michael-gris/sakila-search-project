Финальный проект по курсу Python

Название: Каталог фильмов с аналитикой

Описание:
Программа позволяет выполнять поиск фильмов из базы данных (MySQL) по ключевым словам или по жанру и году выпуска.
Все пользовательские запросы логируются в MongoDB с фиксацией времени, типа запроса и параметров.
Дополнительно реализована возможность отображения последних 5 выполненных запросов.

Функциональность:
- Подключение к базе данных MySQL (Sakila)
- Запросы по частям названий фильмов
- Запросы по ID жанра и диапазону годов
- Подсчёт и отображение топ-5 популярных типов запросов
- Табличный вывод результатов (через библиотеку tabulate)
- Логирование всех запросов в MongoDB
- Просмотр последних 5 запросов по времени
- Работа через консольное меню

Используемые технологии:
- Python 3.12
- MySQL (через mysql-connector-python)
- MongoDB (через pymongo)
- dotenv (переменные окружения)
- tabulate (табличный вывод)
